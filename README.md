Getting Started TO RUN THE APP: 
flutter pub get 
flutter run

IF YOU GET AN ERROR WITH DART VERSION, RUN: 
flutter channel stable 
flutter upgrade 
flutter pub get 
flutter run

IF YOU WANT TO RUN WITH SPESIFIC DEVICE OR EMULATOR, RUN: 
flutter pub get 
flutter run -d <device_id>

YOU CAN GET DEVICE ID WITH: 
flutter devices
